## Platform-Specific Adjustments

<!-- Known issues to watch for, based on past migration sessions -->

| Issue | Platform | Preventive Fix |
|-------|----------|---------------|
| Text vertical centering | iOS | Explicit `lineHeight` matching container; or `paddingBottom: 0.5` |
| Border-radius clipping | Android | `overflow: 'hidden'` on parent AND all children with borderRadius |
| flexDirection default | Both | Always explicit `flexDirection: 'row'` when needed (RN defaults to column) |
| Inset shadow | Both | Remove entirely — no equivalent |
| fit-content width | Both | Use `alignSelf: 'flex-start'` or remove |
