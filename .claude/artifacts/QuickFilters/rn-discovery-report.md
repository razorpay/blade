## CSS Grid Pattern (Space Reservation Trick)

The web renders two stacked `Text` nodes in a CSS Grid cell: an invisible `semibold` clone to pre-reserve horizontal space, and the visible text that switches weight on selection. This prevents layout shift when the container width would otherwise grow/shrink as font-weight changes.

On native, font-weight changes do not cause layout shifts the same way (RN measures text differently), so this pattern is unnecessary. A single `Text` node with dynamic `weight` and `color` is sufficient.

## Animation Requirements
