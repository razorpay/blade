feat(Slider): add Slider component

Adds a new `Slider` component that allows users to select a value from a continuous or discrete range by dragging a thumb along a track.

Features:
- Controlled and uncontrolled usage with `value` / `defaultValue`
- Configurable `min`, `max`, and `step`
- Form integration with `label`, `helpText`, `errorText`, `successText`, and `validationState`
- Three sizes: `small`, `medium`, `large`
- Disabled state with `isDisabled`
- Accessible via proper ARIA attributes
- Optional value display with `showValue`
