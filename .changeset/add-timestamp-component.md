feat(Timestamp): add Timestamp component for standardised date/time display

Adds a new `Timestamp` component that solves the problem of 50+ unique date format strings across dashboards by providing a single, locale-aware API for all date/time display needs. Supports relative ("5 minutes ago"), date-only, time-only, and combined date-time formats with configurable date/time styles, hour cycle, and precision.
