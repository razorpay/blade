# When to use Svelte General Documentation

General documentation for Blade Svelte covers setup and usage fundamentals for Svelte 5 applications.

## Installation

Install `@razorpay/blade-svelte`, import theme CSS and fonts from `@razorpay/blade-core`, and verify your build setup.

## Usage

Import components from `@razorpay/blade-svelte/components`, enable dark mode with `data-theme`, and follow Svelte 5 patterns (runes, snippets).
